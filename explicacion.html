Proyecto final Minervini

proyecto final de felipe minervini para el curso de javascript de coderhouse comicion 49760.

el proyecto se basa en un generador de contraseñas semi aleatrio creado con la funcion de math.random (y una ecuacion que lo complejisa)
ademas de crear la contraseña te permite almacenarla de forma local con los datos de mail y las plataformas, los datos almacenados se pueden ver en la pestaña de listado
